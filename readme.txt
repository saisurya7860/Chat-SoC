# ChatSoC

![ChatSoC Logo](path/to/logo.png)

## Overview

ChatSoC is an AI-powered chatbot developed as a 2nd-year mini project. It integrates multiple features such as login, signup, YouTube downloads, QR code generation, AI image generation, and more using various API keys.

## Features

- **Login and Signup**: User authentication (Note: Login IDs are not encrypted).
- **YouTube Downloads**: Download videos from YouTube.
- **QR Code Generator**: Generate QR codes for various data.
- **AI Image Generator**: Generate images using AI.
- **General Conversation**: Engage in human-like conversations.

## Setup

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/chat-soc.git
    cd chat-soc
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Update necessary file paths, login data, and API keys in the configuration files.

## Usage

1. Run the main application:
    ```bash
    python main.py
    ```

2. Interact with the chatbot using the provided commands:
    - `/login` - Login to your account
    - `/signup` - Create a new account
    - `/youtube` - Download a YouTube video
    - `/qr` - Generate a QR code
    - `/image` - Generate an AI image
    - `/chat` - Start a conversation
